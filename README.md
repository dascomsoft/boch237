# Boch237 – La réussite, simplifiée

##  Le problème que nous résolvons

Au Cameroun, la recherche d’un répétiteur qualifié reste un véritable parcours du combattant. Parents et élèves se tournent encore massivement vers WhatsApp, les groupes Facebook ou le bouche-à-oreille. Résultat : **aucune transparence, aucun contrôle qualité, aucune traçabilité**. Les bonnes rencontres relèvent souvent du hasard, et les répétiteurs sérieux peinent à se faire connaître. Face à ce constat, Boch237 est né.

---

##  Ce qu’apporte Boch237

Boch237 est une **plateforme de mise en relation parents/répétiteurs 100% camerounaise**. Elle permet :

- 🔍 **Trouver un répétiteur** en quelques clics (par province, ville, classe, matière)
-  **Donner de la visibilité aux répétiteurs** qualifiés
- 💬 **Discuter en temps réel** via un chat sécurisé (sans partage de numéro)
-  **Publier des annonces de besoin** de cours (validées par un admin)
- **Dashboard administrateur** pour modérer, valider et gérer la plateforme

Boch237, c’est la **porte d’entrée unique** pour une éducation simplifiée au Cameroun.

## Stack technique utilisée

###  Frontend
- **Next.js 15** – Framework React avec App Router
- **TypeScript** – Typage statique robuste
- **Tailwind CSS** – Stylisation rapide et responsive
- **Lucide React** – Icônes modernes et légères
- **Axios** – Requêtes HTTP simplifiées
- **Socket.IO Client** – Communication temps réel (chat)

### ⚙️ Backend
- **Node.js** – Runtime JavaScript
- **Express** – Framework web léger
- **Socket.IO** – WebSocket pour le chat temps réel
- **MongoDB Atlas** – Base de données cloud gratuite
- **Mongoose** – ODM pour MongoDB
- **JWT** – Authentification sécurisée (JSON Web Token)
- **bcryptjs** – Hachage des mots de passe
